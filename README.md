# DSP Labs

This repository contains Digital Signal Processing (DSP) laboratory assignments and implementations. The labs cover various DSP concepts and practical applications.

## Structure

The repository is organized by lab numbers:

- Lab 1: Basic Python DSP Implementation
- Lab 2-10: Various DSP Concepts and Applications

Each lab folder contains the respective Python scripts and implementations.

## Requirements

- Python 3.x
- NumPy
- Other required libraries (specific to each lab)
